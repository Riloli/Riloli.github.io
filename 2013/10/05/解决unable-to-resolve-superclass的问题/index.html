谷歌后才知道ADT 17以上如果引用Jar文件就需要放在libs目录上…

可是大蛋已经放好了啊.我就奇怪了~这尼玛是神马回事!!

琢磨了一番后才发现我没有勾上编译libs目录的选项!233333

开启的具体步骤如下:

1. 右键项目,选择属性
2. Java Build Patch -> Order and Export
3. 勾上 Android Private Libraries
4. 重新编译即可
